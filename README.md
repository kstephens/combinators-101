# combinators-101

An Interactive Introduction to Function Combinators

Requires Python 3.13.

Run:

```bash
scripts/setup.sh
. venv/bin/activate
```

Open notebooks:

```bash
open c101/combinators_101.ipynb
```

Before git commit:

```bash
scripts/before-commit.sh
```

Run the notebook code directly:

```bash
scripts/before-commit.sh
python c101/combinators_101.py
```

View the notebooks:

```bash
open html/*/*.html
```
