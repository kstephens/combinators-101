# combinators-101

An Interactive Introduction to Function Combinators

Run:

```bash
./scripts/setup.py
. venv/bin/activate
```

Open notebook:

```bash
open c101/combinators-101.ipynb
```

Before git commit:

```bash
./scripts/before-commit.sh
```

Run the notebook code directly:

```bash
./script/sbefore-commit.sh
python combinators-101.py
```
