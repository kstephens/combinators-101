# combinators-101

An Interactive Introduction to Function Combinators

Run:

```bash
./setup.py
. venv/bin/activate
```

Open notebook:

```bash
open ./combinators-101.ipynb
```

Before git commit:

```bash
./before-commit.sh
```

Run the notebook code directly:

```bash
./before-commit.sh
python combinators-101.py
```
