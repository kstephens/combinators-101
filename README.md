# combinators-101

An Interactive Introduction to Function Combinators

Requires Python 3.13.

Run:

```bash
bin/c101 setup
. venv/bin/activate
```

Open notebooks:

```bash
open c101/combinators_101.ipynb
```

Before git commit:

```bash
bin/c101 precommit
```

Run the notebook code directly:

```bash
bin/c101 convert-to-py
python c101/combinators_101.py
```

View the notebooks:

```bash
open html/*/*.html
```
